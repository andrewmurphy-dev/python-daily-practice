what is the enclosing scope ?

#the word encclosing means surrounding

#what is scope in this context ?

this is the region of code where python can find and use a particular name !  


this is the scope of an outer function that surrounds the inner function 

#for example 


def outer():
    message = "Hello"  # enclosing variable for inner()

    def inner():
        print(message)

    inner()

outer()




#message is local to outer(), but from inner()’s perspective, it is in the enclosing scope.

#think of functions as Boxes !

outer()

│ ┌─────────────────────────┐ │
